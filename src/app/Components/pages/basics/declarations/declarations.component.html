• let

    El ámbito de una variable declarada con let es en aquellas sentencias que básicamente están encerradas entre llaves sean definiciones de funciones o bloques if, while, for y loops similares.

  • const

    El ámbito o scope para una variable declarada con const es, al igual que con let, en bloque, pero una declaración con let previene la sobreescritura de variables, const directamente prohíbe la reasignación de valores.
